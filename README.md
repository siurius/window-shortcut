# window-shortcut

## Preparation

1. wmctrl
2. xbindkeys
    - `xbindkeys -d > ~/.xbindkeysrc`

## Usage

### Set shortcut to raise a window

1. `./ws.sh`
2. Input the number of selected window
3. Press shortcut

### Remove all shortcut

Shortcuts will be reset after reboot. You can still remove them use `./ws.sh -r`
